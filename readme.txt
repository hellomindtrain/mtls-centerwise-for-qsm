A powerful extension for Quiz And Survey Master (QSM) that adds Multi-Center and Franchise management capabilities.

== Description ==

MTLS Centerwise empowers computer institutes, coaching centers, and schools to run a franchise model using QSM. 
It segregates QSM quiz results center-wise so that every Center Manager only sees the results of their own enrolled students.

= Features of Lite Version =
* QSM Result Segregation: Automatically fetches QSM quiz results and filters them strictly center-wise.
* Franchise Data Privacy: Center Managers can ONLY view the exam results and profiles of students enrolled in their own center.
* Superadmin Global View: Site administrators get a centralized dashboard to monitor system-wide statistics, all enrolled students, and all QSM results across every branch.
* Branch Management: Superadmins can register new center branches and remove centers directly from the frontend.
* Student Assignment: Seamlessly map WordPress users (students) to specific branches/centers for targeted QSM tracking.
* Dedicated beautiful Frontend Dashboard for Center Owners to monitor their branch's QSM results.
* Dedicated Frontend Profile & Result Dashboard for Students.

= PRO Features =
To unlock Advanced Dashboard Analytics, Real-time Search & Filtering, Detailed Student Table Views, Exact Time Tracking (MM:SS) for quizzes and more features, please upgrade to the [PRO Version](https://mtls.tech/centerwiseproforqsm).

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory, or install the plugin directly through the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Follow the Setup Guide inside the "MTLS Centerwise" menu on your admin dashboard.
4. Create a page and use the shortcode `[mtls_qsm_dashboard]` for students and center owners.
5. Create a hidden page and use the shortcode `[mtls_qsm_super_admin_dashboard]` for administrative controls.
